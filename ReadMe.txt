Инструкция по расширенной установке для проекта LoliLand.net;

Что такое "Расширенная установка"?
Простыми словами - это замена файлов перевода для некоторых модификаций, чтобы наш основной набор UniversalNew смог изменять перевод в них.

Зачем нужна "Расширенная установка"?
Буквально - она нужна только тем, кто хочет получить более качественный перевод на Русский язык, так-как мы, Lena'S OPG тоже этим занимаемся.

А что будет, если не выполнить "Расширенную установку"?
Почти ничего, для обычной установки текстур-пака достаточно просто перекинуть .zip архив в папку resourcepacks.

Инструкция:
1. Откройте архив UniversalNew.zip любым архиватором (Проводник или WinRAR);
2. Перенесите папку lang на свой пк (Например, на рабочий стол). РАСПАКОВЫВАТЬ ВЕСЬ НАБОР НЕ НУЖНО!!!;
3. Откройте корневую папку сборки, на которой Вы играете;
4. Перенесите свежую папку lang в корневую папку сборки;
5. Согласитесь на замену файлов и немного подождите;
6. Готово. Рекомендуем перезапустить клиент для полного применения изменений.

ВНИМАНИЕ: В папке resourcepacks должны быть ТОЛЬКО АРХИВЫ, НЕ НУЖНО РАСПАКОВЫВАТЬ ВЕСЬ НАБОР, ИНАЧЕ РАБОТАТЬ НИЧЕГО НЕ БУДЕТ!!!
